## Overview

Welcome to my portfolio website — a space that showcases my journey and growth as a software developer. The site is home to a curated collection of projects that reflect my skills in React, Node.js, JavaScript, and MongoDB. Each project highlights my ability to build interactive, responsive, and user-centric web applications.

Beyond project listings, the portfolio provides insight into my development philosophy and design aesthetics. Whether you're an employer, client, or collaborator, this platform is designed to help you explore my technical capabilities and creative approach. Feel free to browse and get in touch!

## Key Features
- **Project Showcases**: Detailed write-ups of featured projects, including tech stacks, outcomes, and live links.
- **Clean, Responsive Design**: Built for performance and accessibility, with a mobile-first layout and visually engaging interface.
- **About Me Section**: Highlights my journey, skills, and interests in software engineering and product design.
- **Contact Integration**: Visitors can reach out directly via a dynamic contact form without backend complexity.
- **Live Deployments**: Projects are deployed and linked for real-time exploration by potential collaborators or recruiters.

## Technologies Used
- **Frontend**:  
  - React.js  
  - Tailwind CSS / Custom Styling  
  - React Icons for visual cues  

- **Backend (minimal)**:  
  - Static-hosted — no backend required for portfolio content  
  - EmailJS used for contact form functionality without a server  

- **Deployment**:  
  - Hosted on **Netlify** for fast and global performance  

## Services Used

- **EmailJS**:  
  Used for sending form submissions directly to email without a backend. This integration keeps the site lightweight while still enabling communication with visitors.

## Challenges and Learnings

Designing the portfolio meant balancing minimalism with expressive design. It was important to create a space that felt personal while remaining professional. Implementing email functionality without a backend using EmailJS also helped streamline the setup while maintaining user experience and form reliability. Mobile responsiveness and load optimization were key focus areas during the build process.


## Outcome

This portfolio acts as a digital footprint of my development journey — bringing together both functionality and storytelling. It serves as a central platform to connect with potential clients, recruiters, and collaborators, while showcasing my passion for web development and modern design.
